父亲节 Father's Day

我们通过与自己的父亲融合，才能找到灵性的归路

Juny 20, 2021
  • 0
  • 0
  • 0
  • 0

允许小孩模仿自己的父母， 小孩才有机会挣脱父母的“魔咒”并从中成长

在父亲的手中，孩子才能够获得通往这个世界的道路。妈妈是做不到的。

我们通过与自己的父亲融合，才能找到灵性的归路。

人必须要有勇气才能得到幸福。

没有爱，就没有生命。每一个人都是自己的“灵”。

人只要放下恐惧，眼前就会出现道路，最后一路平安。

当一个人能与自己和谐相处，爱与尊重自己的父母，忧郁就会随之消失。可是我得承认，这是非常困难的，所以如果你能成功，就会体验到一份恩赐。

生命的意义在于单独的个体如何应对早已被安排的命运。

生命消逝， 就如同树叶飘落， 将空间让给新芽发展， 虽然已化为尘土， 却仍滋养着现存的生命。

人是有力量解决问题的

很惭愧，至今为止我做的还很不够，我还在学习，学习各种心理学、育儿知识、教育学、传记..... 学习做一个真正的爸爸。

做爸爸也是一份工作，有重大的责任。但也是一种救赎，没有孩子给我机会，我哪里来出演“爸爸”这个角色的机会呢？要知道生命仅仅存在于和他人的经历中以及你用何种方式与他们接触和碰撞（Your life has nothing to do with you. It is about every one whose life you touch and how you touch it.)

爸爸这份工作主要两件任务：情感连接＋权力超越。

1）情感连接是给孩子爱、鼓励、温暖、光。孩子是独立的生命，他们是与父母完全不同的独立个体，孩子的生命存在本身就是最大价值，并不因他们做了什么事，取得了什么成就，不是因为他们的不凡，才能赢得我们的什么。抛开我们对孩子功利的需索，我们会与孩子建立深厚的联结以及积极的正向的关系。而这种关系和情感的建立，是孩子获得自己存在的“正当感”的依托。他们可以不必依赖外界的认同与品评，建立自己强健的精神世界。关系与情感胜过一切。

2）权力超越是帮助孩子成为独立的人格，把人生权力交给孩子，和帮助她飞的更远。给孩子做自己的权力，不应试图改变她，接受她本来的样子，引导她的行为，培育她特殊的个性，以使这些气质有助于她将来的发展，而不是成为她的阻碍。

这些年，我在不断的纠正自己，我也曾犯了一些错误。通过反省我发现，穿越了真实的不完美的自己和成人的自负感。就像镜子，照出我的问题和真正的自己。我要感谢孩子，孩子是大人啊，是朋友、老师。

孩子是朋友，可以交流的朋友，可以放松的状态相处的朋友，相互帮助的朋友。

孩子是老师，像一面镜子，发现自己的丑和美。让我内心成长。接纳孩子的程度同接纳自己的程度成正比，接纳孩子，就是接纳自己。挑剔孩子，实则挑剔的也是我们自己。如何应对孩子的错误，事实上也是我们对自身错误的应对。从这个意义上讲，孩子对我们的教益和帮助，可能远远胜过我们对孩子的贡献。

哲学家说，当一个人放下所有的预期和目的，以自己的本真与另一个人建立关系时，就会与另一个人的本真相遇。或许这种相遇的境界太高，但是作为普通人，父母至少可以学习做到尊重孩子是一个独立的生命，而不是父母的延续，更不是实现父母想象中那个“美好的孩子”。父母抛开附加在孩子身上的那些要求、期许和目的的过程中，也会重新认识自己，看见潜意识中真实的自己。

“我被你的存在感染了。你的一部分在我之内生长和成长。” 很美的话。

感恩！要加油！

《渴望生活 • 梵高传》

一个因善良、真实的天使在人间痛苦磨难，渴望生活

Juny 21, 2021
  • 0
  • 0
  • 0
  • 0

《梵高传 • 渴望生活》记录了一个单纯温柔却强有力的男子，他质朴、炽烈，拧着一股与世俗死磕的癫狂，他把渴望融化成那抹纯粹的湛蓝或者金黄，燃烧陨落直至消亡......世人不懂你，因为 Vincent，the world is never meant for one as beautiful as you.

文森特·威廉·梵高（Vincent Willem van Gogh，1853年3月30日—1890年7月29日），荷兰后印象派画家。代表作有《星月夜》、自画像系列、向日葵系列等。

梵高出生于1853年3月30日荷兰乡村津德尔特的一个新教牧师家庭，早年的他做过职员和商行经纪人，还当过矿区的传教士最后他投身于绘画。他早期画风写实，受到荷兰传统绘画及法国写实主义画派的影响。1886年，他来到巴黎，结识印象派和新印象派画家，并接触到日本浮世绘的作品，视野的扩展使其画风巨变。1888年，来到法国南部小镇阿尔，创作《阿尔的吊桥》；同年与画家保罗·高更交往，但由于二人性格的冲突和观念的分歧，合作很快便告失败。此后，梵高的疯病（有人记载是“癫痫病”）时常发作，但神志清醒时他仍然坚持作画。1889年创作《星月夜》。1890年7月，梵高在精神错乱中开枪自杀，年仅37岁。

温森特 梵高一生都是苦难，画画是他唯一的救赎，却过着饥饿困苦食不饱腹、被人嘲笑、不被理解、没有收入的日子，也没有一个女人愿意嫁给他，即使有也被家族阻止，因为梵高没有收入。全靠梵高的弟弟(提奥 梵高)一直救济他，每月寄钱150法郎，还骗他说不是救济，是帮梵高卖画的钱，其实梵高的画一幅都不曾卖出过。后来，梵高发现了，他弟弟病了，也快失业了，梵高不想再连累弟弟，也不想连累弟媳和弟弟的孩子，他觉得一切应该结束了，就开枪自杀了。

梵高绘画艺术的真实感及其所表现出来的对生命力的追求及对灾难民众不屈于命运的关切之情，我们会为之一震。由于这种面对社会丑恶、腐朽而表现出的不屈的肉体将鼓舞后来者在艺术长河中不时地探究与追求。论及个人，梵高无疑是不幸的，他单纯的理想主义信心，痛苦的疾病折磨，孤僻、直率、执着而神经质的性格使他难以融入当时的社会，并与人为善相处。或许正是这种性格缺陷成就了梵高的不朽。他以广博的文学涵养，宽厚的人文情怀，博大的同情心，悲悯的基督肉体，炽热的创作激情，共同的绘画技巧，把生命中最珍贵的东西熄灭在绘画艺术中，构成了他得以不朽的艺术生命……

一个因善良、真实而受苦天使在人间的痛苦磨难、渴望生活、直到开枪自杀的一生，善良、高尚、炙热的灵魂，因色彩而享乐的天才，画在了他的画里。

一生渴望生活，悲悯世人的梵高抬头看漆黑的夜幕，望见了那流动的，炫美的，生生不息的星空，那个时候，他的内心一定被大自然的美深深感动。

梵高曾说，当我画太阳时，我希望使人们感觉到它是在以一种惊人的速度旋转着，正在发出威力巨大的光和热的浪。

当我画一块麦田时，我希望人们感觉到，麦粒内部的原子正朝着它们最后的成熟和绽开而努力。

当我画一颗苹果树时，我希望人们能感受苹果里面的果汁正把果皮撑开，果核中的种子正在为结出自己的果实而努力！

当我给一个男人画像时，我希望人们感觉到这个男人汩汩流过的一生，

——他所见过的一切，他所做过的一切和他所经受过的一切！正如周国平所言，梵高是一个因善良受苦的天使，一个用色彩享乐的天才。

从表面上看，梵高的一生都是在生存，而不是在生活。因为在很多时候任何的一根稻草都有可能将他压垮。但是从深处看，他放弃家境能够给他提供的安逸舒适的日子，就是为了寻找能够为之而活的东西。活着而不能做自己喜欢的事情，与死亡还有什么区别呢。所以他选择了生活的颠沛流离，以获得创作的源泉。当他的作品达到巅峰的同时，他的生命也为之耗尽。

他有多热爱这个世界，热爱生活，就能对它感到多绝望。许多人都爱生活的光明美好，谁不爱呢？可同时又因生活的苦难而备受煎熬……我想也许正是他对生活的美好爱得热烈，所以无法仅仅只是麻木的活下去。

—— “当我画一个太阳，我希望人们感觉到它在以惊人的速度旋转，正在发出骇人的光热巨浪；当我画一片麦田，我希望人们感到麦子正朝着它们最后的成熟和绽放努力；当我画一棵苹果树，我希望人们感觉到苹果内的果汁正把果皮撑开，果核内的种子正在为结出果实而奋起。 当我画一个男人，我就要画出他滔滔的一生。如果生活中不再有某种无限的，深刻的，真实的东西，我也将不再眷恋人间。‘’ ( 梵高《亲爱的提奥》）
