父亲节 Father's Day

我们通过与自己的父亲融合，才能找到灵性的归路

Juny 20, 2021
  • 0
  • 0
  • 0
  • 0

允许小孩模仿自己的父母， 小孩才有机会挣脱父母的“魔咒”并从中成长

在父亲的手中，孩子才能够获得通往这个世界的道路。妈妈是做不到的。

我们通过与自己的父亲融合，才能找到灵性的归路。

人必须要有勇气才能得到幸福。

没有爱，就没有生命。每一个人都是自己的“灵”。

人只要放下恐惧，眼前就会出现道路，最后一路平安。

当一个人能与自己和谐相处，爱与尊重自己的父母，忧郁就会随之消失。可是我得承认，这是非常困难的，所以如果你能成功，就会体验到一份恩赐。

生命的意义在于单独的个体如何应对早已被安排的命运。

生命消逝， 就如同树叶飘落， 将空间让给新芽发展， 虽然已化为尘土， 却仍滋养着现存的生命。

人是有力量解决问题的
