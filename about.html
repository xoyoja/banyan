一个普通人

Julyy 24, 2021
  • 0
  • 0
  • 0
  • 0


生活可以为了许多，但我觉得最终是为了美，但这种美不是一种意义与目的，而是一种持久的快感。

就像纪伯伦思想的骆驼队虽然走不到沙漠的尽头，但心中永远有一处绿洲。

人是一座桥梁，而不是目的，也可以是目的，但怎样都不是手段。

不管是怎样，即使超越的力量与价值无关，但前方永远属于生命，而不是死亡。
